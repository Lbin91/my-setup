## 이 프로젝트는?

Claude Code에서 사용할 수 있는 **다양한 플러그인들을 분석하고 문서화**하는 프로젝트입니다.

- 새로운 플러그인을 발견하면 **쉽고 자세하게** 정리합니다
- **고등학생도 이해할 수 있도록** 친절하게 설명합니다
- ASCII 아트, 표, 비유를 활용해 **가독성 좋게** 작성합니다

## 📝 문서 작성 원칙

### 1. 가독성
- ASCII 아트와 박스로 시각적 구분
- 표로 정보 정리
- 이모지로 카테고리 구분

### 2. 이해하기 쉽게
- 전문 용어는 비유로 설명
- 실제 사용 예시 포함
- 단계별 흐름도 제공

### 3. 정보 기록
- 작성일 기준 버전 명시
- GitHub 저장소 링크 포함
- 라이선스 정보 기재

---

## 🚀 새 플러그인 추가하기

새로운 Claude Code 플러그인을 발견하면:

1. 플러그인의 GitHub 저장소 분석
2. 기존 문서와 동일한 스타일로 작성
3. `claude-code/plugins/` 디렉토리에 `[플러그인명]-guide.md` 형식으로 저장
4. 이 README.md의 플러그인 목록에 추가
